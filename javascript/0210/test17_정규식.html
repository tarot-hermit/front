정규식(Regular Expression)

    형식 : /~~~~~/플러그인 , /[~~~~~]/플러그인 ,       자바 : ^[~~~~]*$
    플러그인(플래그) : 
      g(global) : 전체범위(문자열내의 모든 패턴 검색)에서의 처리(기본)
        예) str.match(/a/); - 최초에 발견된 문자만 반환
            str.match(/a/g) - 모든 문자에서 검색후 결과를 배열로 반환
      m(multi line) : 여러줄에 걸쳐서 처리(textarea 태그)
      s(single line) : 한줄에서의 처리(기본값) - 개행문자(\n)도 포함해서 처리한다.
      i : 영문자인경우 대/소문자를 구별하지 않는다.
      u(unicode) : 유니코드에 대한 구별처리시 사용

      정규식을 실행하기위해서 사용되는 함수?
        1. 사용자입력문자열.match(정규표현식 or 정규표현식담은변수) : 포함유무에 대한 비교시에 사용할때 편리함.
        2. 정규표현식변수.test(사용자입력문자열) : 구체적인 문자(열)간의 비교 체크시에 사용하면 편리함.

====================================================================================================
메서드                                          의미
====================================================================================================
("문자열").match(/정규표현식/플래그)              "문자열"에서 "정규표현식"에 매칭되는 항목들을 배열로 반환
----------------------------------------------------------------------------------------------------
("문자열").replace(/정규표현식/, "대체문자열")      "정규표현식"에 매칭되는 항목을 "대체문자열"로 변환
----------------------------------------------------------------------------------------------------
("문자열").split(정규표현식)                    "문자열"을 "정규표현식"에 매칭되는 항목으로 쪼개어 배열로 반환
----------------------------------------------------------------------------------------------------
(정규표현식).test("문자열")                     "문자열"이 "정규표현식"과 매칭되면 true, 아니면 false반환
----------------------------------------------------------------------------------------------------
(정규표현식).exec("문자열")                     match메서드와 유사(단, 무조건 첫번째 매칭 결과만 반환)
----------------------------------------------------------------------------------------------------
